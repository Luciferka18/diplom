Stage 12 — detail pages v2

Applies frontend-only redesign for:
- /shop/[id]
- /programs/[id]
- /articles/[id]
- /trainers/[id]

Both light and dark themes are supported through CSS variables and dark: classes.

How to apply:
1. Extract into D:\diplom
2. Run nashfit_detail_pages_stage12\APPLY_DETAIL_PAGES_STAGE12.bat
3. Restart frontend:
   cd D:\diplom\my-app
   npm run dev
4. Refresh pages with Ctrl+F5

No backend files, database, .env, node_modules or vendor are changed.
