NashFit Account Today Real Hotfix

Fixes /account Today block:
- removes fake static workout/streak/goal cards
- shows only real account data: active program, upcoming booking, latest order, unread notifications
- shows honest empty state when there is no real data

Install:
1. Extract into D:\diplom
2. Run nashfit_account_today_real_hotfix\APPLY_ACCOUNT_TODAY_REAL_HOTFIX.bat
3. Restart frontend: cd D:\diplom\my-app && npm run dev
4. Refresh /account with Ctrl+F5
