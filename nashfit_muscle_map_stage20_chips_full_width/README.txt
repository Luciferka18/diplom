NASHFIT MUSCLE MAP STAGE20 CHIPS FULL WIDTH

Что исправлено:
- блок "Быстрый выбор мышцы" вынес из левой колонки;
- теперь он растягивается на всю ширину основной карточки, включая правую часть;
- кнопки мышц идут ровной сеткой: на desktop 7 колонок и 2 строки;
- SVG, подсветка, backend и блок упражнений сохранены из предыдущей версии.

Главный изменённый файл:
- my-app/src/components/muscles/MuscleMap.jsx

Как установить:
1. Распакуйте папку nashfit_muscle_map_stage20_chips_full_width в корень проекта, где лежат my-app и fitlab.
2. Запустите APPLY_MUSCLE_MAP_STAGE20_CHIPS_FULL_WIDTH.bat
3. Перезапустите фронт:
   cd my-app
   npm run dev

Откат:
ROLLBACK_MUSCLE_MAP_STAGE20_CHIPS_FULL_WIDTH.bat
