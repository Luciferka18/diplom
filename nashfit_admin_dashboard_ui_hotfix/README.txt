Admin dashboard UI hotfix

Исправляет читаемость KPI-карточек в светлой теме и переносит блок быстрых действий выше статистики.

Установка:
1. Распаковать архив в D:\diplom
2. Запустить admin_dashboard_ui_hotfix\APPLY_ADMIN_DASHBOARD_UI_HOTFIX.bat
3. Перезапустить frontend: cd D:\diplom\my-app && npm run dev
