Как установить:
1. Распакуйте папку nashfit_home_redesign_stage13 в корень проекта,
   где лежат папки my-app и fitlab.
2. Запустите APPLY_HOME_REDESIGN_STAGE13.bat
3. Перезапустите фронт:
   cd my-app
   npm run dev
