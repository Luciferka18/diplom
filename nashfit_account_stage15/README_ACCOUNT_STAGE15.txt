NashFit Account Stage 15

What it changes:
- Replaces /account with a user cabinet v2.
- Adds quick actions, weekly progress, today panel, upcoming booking, notifications preview.
- Keeps active membership, program progress, recommendations and activity timeline.
- Light and dark themes are handled through CSS variables and dark: classes.

Install:
1. Extract nashfit_account_stage15 into D:\diplom
2. Run nashfit_account_stage15\APPLY_ACCOUNT_STAGE15.bat
3. Restart frontend: cd D:\diplom\my-app && npm run dev
4. Open /account and refresh with Ctrl+F5

Does not touch Laravel, database, .env, vendor or node_modules.
