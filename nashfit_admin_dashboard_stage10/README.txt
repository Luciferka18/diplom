NashFit Stage 10 — Admin Dashboard v2

Применение:
1. Распаковать архив в D:\diplom
2. Запустить nashfit_admin_dashboard_stage10\APPLY_ADMIN_DASHBOARD_STAGE10.bat
3. Перезапустить Laravel и Next.js
4. Открыть /admin

Патч добавляет:
- API /api/admin/dashboard/overview
- новый живой dashboard администратора
- графики за 7 дней без внешних библиотек
- последние заказы, записи, активности
- блок проблем базы с подсказкой nashfit:qa --fix

База и .env не очищаются и не изменяются.
