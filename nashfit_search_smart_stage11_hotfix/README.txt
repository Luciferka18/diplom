NashFit smart search hotfix

Что делает:
- заменяет строгий поиск на гибкий клиентский поиск по формам слов;
- добавляет автопредложение при вводе;
- заменяет статичные подсказки на последние запросы, если они есть;
- если последних запросов нет, показывает популярные запросы;
- перерабатывает информационные карточки под реальные данные поиска.

Установка:
1. Распаковать архив в D:\diplom
2. Запустить search_smart_stage11_hotfix\APPLY_SEARCH_SMART_STAGE11_HOTFIX.bat
3. Перезапустить frontend: cd D:\diplom\my-app && npm run dev
