NashFit Stage 11: site search

Добавляет страницу /search и пункт "Поиск" в верхнее меню.
Патч frontend-only: Laravel, база, .env, пользователи и данные не трогаются.

Установка:
1. Распаковать архив в D:\diplom.
2. Запустить nashfit_site_search_stage11\APPLY_SITE_SEARCH_STAGE11.bat.
3. Перезапустить frontend: cd D:\diplom\my-app && npm run dev.
4. Открыть /search.
